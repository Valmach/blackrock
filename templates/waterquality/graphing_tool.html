Using the Graphing Tool

To graph a set of data, follow the 3 steps listed below:

  1. First, enter a title for your graph and select the graph type you're wanting to generate.
  2. Select the data series that you'd like to graph with start/end dates and threshold labels.
  3. Click the green create graph button to see the results.

Note that you can print the graph or download it, too. The icons are located in the top right corner of the graph. The graph can be downloaded in these four formats: PNG, JPEG, PDF, SVG.
