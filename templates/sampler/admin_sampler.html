How To Format CSV Data for Import

To import tree data from a CSV file, please ensure that your file has the following headings: "ID", "x", "y", "species" and "dbh". ID is the tree's unique ID number, species is its species code (e.g. "ro"), "dbh" is its diameter breast height (in cm), and x and y are its location coordinates (in meters).

CSV (comma-separated value) files are just plain-text versions of Excel spreadsheets. The exact process varies depending on your software and version, but you can convert your Excel spreadsheet into a CSV file by selecting "Save As..." and changing the file type (look for the choice with the file extension "*.csv").
